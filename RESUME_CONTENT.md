# Project: Smart AI Expense Tracker

**Technologies:** React.js, Firebase, Material-UI, Web Speech API.

**Key Highlights:**
1.  **AI-Powered Voice Entry:** Implemented **Web Speech API** to enable hands-free transaction logging with custom natural language parsing for Indian numbering systems (Lakhs, Crores).
2.  **Secure Real-Time Architecture:** Built a scalable backend using **Firebase Authentication** (Google OAuth) and **Cloud Firestore**, ensuring secure user data isolation and instant synchronization across devices.
3.  **Modern UI & Automation:** Designed a responsive, dark-mode enabled interface with **Material-UI** and engineered an automated engine for handling recurring payments and budget alerts.
